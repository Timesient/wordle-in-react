# Wordle in React
本项目是对文字游戏 [Wordle](https://www.nytimes.com/games/wordle/index.html) 的练习性质的仿制项目，完成了以下练习目标：
- 主要使用`react`和`react-redux`工具库进行开发
- 相同功能的模块提取为组件并复用
- 像素级还原页面元素的布局、样式和动画效果

在线展示地址：[wordle.dd1969.xyz](https://wordle.dd1969.xyz)
